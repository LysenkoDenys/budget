# budget

### The app to manage your cash flows.

### The application is deployed here:

https://budget4u.vercel.app

<ul>
<li>⭐collect your cash transactions</li>
<li>⭐watch statistics</li>
<li>⭐make plans (fill transactions on future dates)</li>
</ul>

<li>npm install</li>
